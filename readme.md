Jargon/Nomenclature

- #: "pound", "hash", "hashtag" 
- []: brackets, braces, square brackets
- {}: curly braces, curlies, braces
- $: bling, cash, money sign, dollar sign
- !: bang, not (in conditionals)
- (): parentheses, parens
- *: star, splat
- /: whack, forward slash, slash
- \: back slash
- .: dot
- NaN: bread
- `: backtick, tick
